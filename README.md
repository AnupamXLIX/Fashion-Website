# Fashion-Website
A practice project built using:
- HTML5
- CSS
- Flexbox
- Google Fonts

Features

An e-commerce landing page homepage 

- Responsive Layout using Flexbox
- Navigation bar with product categories
- Hover effects on images and menu items
- Promotion Banner Section
- Festival Sales table
- Structured Footer with informational content
- Clean typography using Google Fonts

This project was made to practice navigation design, layout structuring on the page, Hover animation and Resonsive design fundamentals overall.
